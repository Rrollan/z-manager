# Z Manager

Z Manager is an elegant, high-performance desktop application for managing AI accounts, tokens, and quotas. Built with Tauri, React, and Tailwind CSS.

## Features

- **Account Management**: Seamlessly add, manage, and switch between various AI provider accounts.
- **ZCode Integration**: Specialized support for Z.ai and ZCode accounts, including seamless IDE session switching.
- **Quota Tracking**: Real-time quota tracking and usage monitoring across multiple providers.
- **Cross-Platform**: Available on macOS and Windows with native system integrations.
- **Light/Dark Mode**: A beautiful, responsive interface that respects your system preferences.

## Installation

Download the latest release for your platform from the [Releases](https://github.com/Rrollan/z-manager/releases) page.

### Supported Platforms
- macOS (Apple Silicon & Intel)
- Windows (x64)

## Development

Z Manager is built with [Tauri](https://tauri.app/).

### Prerequisites
- Node.js 20+
- Rust toolchain
- Tauri CLI prerequisites for your OS

### Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run tauri dev
   ```

3. Build the application:
   ```bash
   npm run tauri build
   ```
